Manual Server Update Commands
================================

To update the server at atoz@192.168.0.201, SSH into the server and run these commands:

1. Connect to the server:
   ssh atoz@192.168.0.201

2. Navigate to the application directory:
   cd /volume1/docker/rfms-uploader

3. Pull latest code from Git:
   export HOME=/volume1/docker/rfms-uploader
   /bin/git pull origin main

4. Stop the container:
   /usr/local/bin/docker-compose -f docker-compose-nas.yml down

5. Rebuild the Docker image:
   /usr/local/bin/docker-compose -f docker-compose-nas.yml build --no-cache

6. Start the container:
   /usr/local/bin/docker-compose -f docker-compose-nas.yml up -d

7. Check status:
   /usr/local/bin/docker-compose -f docker-compose-nas.yml ps

8. View logs (optional):
   /usr/local/bin/docker-compose -f docker-compose-nas.yml logs -f

Alternative: Quick restart (if only restarting without code changes):
   /usr/local/bin/docker-compose -f docker-compose-nas.yml restart uploader
